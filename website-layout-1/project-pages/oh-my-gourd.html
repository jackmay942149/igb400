Description

This project was made for a game jam given the prompt "Chain Reaction". The game jam lasted for one week and in the three weeks prior we worked on getting the rendering and vr integration working. It is a simple game where the players is set in a graveyard amongst a murder of menacing pumpkins. The player can explode pumpkins, and exploding pumpkins into others will cause them to explode. The player must wear caution as failure to cause the chain reaction will cause more pumpkins to appear.

Team Information

The team for this project consisted of 2 programmes, including myself, and an artist onboarded in the final week.
