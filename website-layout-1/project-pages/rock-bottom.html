Description

This project is an on-going project with INAP to help educate university students on the danger of mismanaged mining waste causing AMD. We were tasked to create a game for this purpose and decided that a mobile game would be a accessible option given the target audience. It has been release on the Android Play Store and is receiving updates as we begin an iOS port. The game is a 3D puzzle game which uses the accelerometer to guide water balls down a mine site.

Team Information

This was a collaborative effort that had 1 Programmer (myself), 2 Designers, 1 Artist and 2 Consultants (Rio Tinto & BHP Representative).
